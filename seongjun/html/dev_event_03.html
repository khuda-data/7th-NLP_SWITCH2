AWSKRUG 플랫폼엔지니어링 모임 Agentic MCP for EKS Hands-on Lab (7월1일)

Hosted By
류한진 and Subin K.

Details

AWSKRUG 플랫폼엔지니어링 밋업이 열립니다. 이번 밋업에서는 AI 에이전트로 어떻게 쿠버네티스 플랫폼을 운영을 활용할 수 있는지 핸즈온으로 알아봅니다.

■ 일시 및 장소 2025년 7월 1일 화요일 19:00 ~ 서울 강남구 테헤란로 231 센터필드 EAST 18층 18.300, 18.301

■ 발표 주제 Agentic MCP for Amazon EKS 핸즈온: Gen AI 기반 인프라 운영의 새로운 패러다임 - 최우형 프린시펄 솔루션즈 아키텍트, AWS

Gen AI가 코드만이 아닌 운영 방식 자체를 혁신하는 시대 , Amazon EKS 위에서 실행되는 Agentic MCP(Model Context Protocol)는 인프라 운영의 새로운 가능성을 제시합니다.

본 워크샵에서는 오픈소스로 공개된 Agentic MCP for EKS 핸즈온랩 실습을 통해,

  • Amazon Q Developer 와 연계된 AI 에이전트가 어떻게 Kubernetes 기반 운영을 지원할 수 있는지,
  • 코드 생성 + 실행 + 트러블슈팅 이 하나의 흐름으로 연결되는 Agentic Workflow가 어떤 모습인지,
  • 실제 Git 기반 설정 파일과 EKS 리소스를 자동으로 생성·관리하는 방식

등을 실습 중심으로 알아봅니다.

Infra-as-Code의 다음 진화 단계에 관심 있는 AWS 사용자라면, Gen AI 시대에 맞는 DevOps의 미래를 함께 체험해 보세요.

핸즈온을 위해 랩탑 지참이 필요합니다.

■ 참가 안내

  • 밋업 신청을 하시면서 건물출입 등록정보를 입력 후, 참가회비 10,000원을 [카카오뱅크 7979-88-82873 류한진]에 입금하시면, 추후 입금 확인 후 등록이 완료됩니다.
  • 참가회비는 간단한 식사 제공 및 모임 활동에 이용됩니다.
  • 입금하신 분들에게 밋업 신청시 입력하신 휴대폰 번호로 센터필드 출입QR문자가 전송될 예정입니다. 건물 출입 등록을 위하여, 밋업 신청시 반드시 정보를 입력 부탁드립니다. (해당 정보는 건물 출입 등록에만 이용됩니다.)
  • 회비 입금 및 참가신청은 2일전까지 받고 있으며 환불은 1일 전까지 가능합니다. 입금 확인 및 처리는 이벤트 1일전에 배치로 처리하고 있습니다. 양해 부탁드립니다.
