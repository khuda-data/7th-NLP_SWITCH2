쿠키 기본 설정 선택

당사는 사이트와 서비스를 제공하는 데 필요한 필수 쿠키 및 유사한 도구를 사용합니다. 고객이 사이트를 어떻게 사용하는지 파악하고 개선할 수 있도록 성능 쿠키를 사용해 익명의 통계를 수집합니다. 필수 쿠키는 비활성화할 수 없지만 '사용자 지정' 또는 ‘거부’를 클릭하여 성능 쿠키를 거부할 수 있습니다. 사용자가 동의하는 경우 AWS와 승인된 제3자도 쿠키를 사용하여 유용한 사이트 기능을 제공하고, 사용자의 기본 설정을 기억하고, 관련 광고를 비롯한 관련 콘텐츠를 표시합니다. 필수가 아닌 모든 쿠키를 수락하거나 거부하려면 ‘수락’ 또는 ‘거부’를 클릭하세요. 더 자세한 내용을 선택하려면 ‘사용자 정의’를 클릭하세요.

쿠키 기본 설정 사용자 지정

AWS는 다음과 같은 목적으로 쿠키 및 유사한 도구(총칭하여 "쿠키")를 사용합니다.

필수

필수 쿠키는 AWS 사이트 및 서비스를 제공하는 데 필요하며, 비활성화할 수 없습니다. 일반적으로 개인 정보 보호 기본 설정, 로그인 또는 양식 작성 등 사이트 내에서 사용자가 수행한 작업에 상응하는 쿠키가 설정됩니다.

성능

성능 쿠키는 AWS에서 사이트 경험 및 성능을 개선할 수 있도록 고객이 AWS 사이트를 탐색하는 방법에 대한 익명의 통계를 제공합니다. 승인된 제3자가 AWS를 대신하여 분석을 수행할 수 있지만, 해당 데이터를 다른 특정 목적으로 사용할 수는 없습니다.

허용됨

기능

기능 쿠키는 유용한 사이트 기능을 제공하고, 사용자의 기본 설정을 기억하며, 관련 콘텐츠를 표시하는 데 도움을 줍니다. 승인된 제3자가 이러한 쿠키를 설정하여 특정 사이트 기능을 제공할 수 있습니다. 이러한 쿠키를 허용하지 않으면 이러한 서비스 중 일부 또는 전체가 제대로 작동하지 않을 수 있습니다.

허용됨

광고

광고 쿠키는 AWS의 광고 파트너가 AWS 사이트를 통해 설정할 수 있으며, 관련 마케팅 콘텐츠를 제공하는 데 도움을 줍니다. 이러한 쿠키를 허용하지 않으면 관련성이 낮은 광고가 표시됩니다.

허용됨

일부 유형의 쿠키를 차단하면 AWS 사이트 경험이 영향을 받을 수 있습니다. 언제든지 이 사이트의 바닥글에서 [쿠키 기본 설정]을 클릭하여 해당하는 쿠키 기본 설정을 변경할 수 있습니다. AWS 사이트에서 AWS 및 승인된 제 3자가 쿠키를 사용하는 방법에 대한 자세한 내용은 AWS 쿠키 공지 사항 을 참조하십시오.

사용자 개인정보 보호 선택

당사는 크로스 컨텍스트 행동 광고를 포함하여 AWS 사이트 및 기타 자산에 사용자의 관심사와 관련된 광고를 표시합니다. 크로스 컨텍스트 행동 광고는 특정 사이트 또는 앱의 데이터를 사용하여 다른 회사의 사이트 또는 앱에서 사용자에게 광고를 게재합니다.

쿠키 또는 유사한 기술을 기반으로 하는 AWS 크로스 컨텍스트 행동 광고를 허용하지 않으려면 아래에서 ‘허용 안 함’ 및 ‘개인정보 보호 선택 저장’을 선택하거나 글로벌 프라이버시 제어(Global Privacy Control)와 같이 법적으로 인정되는 거부 신호가 활성화된 AWS 사이트를 방문하세요. 쿠키를 삭제하거나 다른 브라우저 또는 디바이스에서 이 사이트를 방문하는 경우 다시 선택해야 합니다. 쿠키 및 쿠키 사용 방법에 대한 자세한 내용은 AWS 쿠키 공지 를 참조하세요.

모두 열기

설문 조사를 완료하였으나 크레딧을 제공받지 못한 경우는 어떻게 해야 하나요?

설문 조사 시 '실습 관련 크레딧 제공이 필요하신가요?' 질문에 '아니오' 또는 응답하지 않으신 경우 크레딧이 제공되지 않습니다.

제공받은 크레딧은 어떻게 사용되나요?

AWS 크레딧은 크레딧이 계정에 등록된 달부터의 사용량에 자동으로 적용되어 소진되거나 만료될 때까지 사용됩니다. 자세한 내용은 AWS 크레딧 약관 을 확인해 주시기 바랍니다.

이메일로 제공받은 크레딧을 어떻게 계정에 등록하나요?

다음과 같이 AWS 계정에 프로모션 코드를 등록할 수 있습니다.

1. 청구 및 비용 관리 콘솔의 크레딧 페이지를 엽니다. 2. 크레딧 사용을 선택합니다. 3. 프로모션 코드의 경우 표시된 대로 프로모션 코드를 입력합니다. 4. 보안 코드의 경우 CAPTCHA에 표시된 코드를 입력합니다. 5. 크레딧 사용을 선택합니다.

프로모션 코드가 올바르게 입력되었고, 유효하며, 만료되지 않았고, 이전에 사용하지 않은 경우 AWS 계정에 등록됩니다.

크레딧을 중복으로 제공받을 수 있나요?

아니오. 해당 월의 모든 세션을 통틀어 하나의 계정으로 한 개의 크레딧 바우처만 수령하고 적용할 수 있습니다.

크레딧은 어떻게 제공되나요?

크레딧은 AWS Techcamp 온라인 세션에 등록할 때 사용한 이메일로 전달됩니다.

크레딧은 누구에게 제공되나요?

온라인 세션 참여 후 설문조사를 완료한 고객분들께 크레딧이 제공됩니다. 설문조사 시 크레딧 수령 질문에 ‘예’ 로 표기해 주셔야 합니다.

과정 완료되면 수료증을 발급 받을수 있나요?

수료증은 감사메일과 함께 행사 종료 후 일정 기간 내 (약 1개월)  발송드리고 있습니다.

세션이 무료로 진행되나요?

온라인 세션에서 실습 중 사용하는 AWS 리소스로 인해 일부 비용이 발생할 수 있습니다. 따라서 실습 후에는 사용하지 않는 리소스를 반드시 삭제하시기 바랍니다. 리소스 삭제 방법은 세션 중에 안내해 드리겠습니다.

실습 위한 계정이 제공되나요?

TechCamp에서는 참가자분들 개인 계정으로 실습이 진행되고 있습니다.
